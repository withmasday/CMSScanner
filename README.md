## Usage

Run CMSScanner with python3

```bash
  [!] python3 cmsscanner.py --target=site.com
  [!] python3 cmsscanner.py --file=list.txt
  [!] python3 cmsscanner.py --file=list.txt --thread=30

        --help HELP
        --target Scan Single URL
        --file Scan From File
            --thread Multiproccess Thread
```
